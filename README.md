# 🌦️ Smart Weather Assistant

Smart Weather Assistant is a clean and responsive web application that provides real-time weather information for any city.  
It is designed with a simple UI and focuses on practical weather details useful for daily planning.

🔗 Live Website:  
https://bhattsana.github.io/smart-weather-assistant/

---

## 🚀 Features

- 🌍 Search weather by city name
- 🌡️ Real-time temperature, humidity & wind speed
- ☁️ Weather condition icons (clear, cloudy, rain, etc.)
- 📱 Fully responsive (mobile + desktop friendly)
- ⚡ Fast and lightweight
- 🎯 Beginner-friendly project structure

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- OpenWeatherMap API
- GitHub Pages (for deployment)

## 📂 Project Structure

# 🌦️ Smart Weather Assistant

A smart and user-friendly weather web application that provides real-time weather information using a public Weather API.  
Built with a clean UI and focused on simplicity, accuracy, and smooth user experience.

🔗 Live Demo: https://bhattsana.github.io/smart-weather-assistant/

---

## 🧠 How It Works

1. User enters a city name
2. App fetches real-time weather data using a Weather API
3. Weather details are displayed instantly on the screen
4. Invalid city names are handled gracefully with proper messages
